# vue-general-components

> 基于 vuepress 文档的 vue 组件快速开发方案

## 项目安装

```
mkdir <project-name>

# 或 npx create-vgc
yarn create vgc

# 或 npm install
yarn install

# 安装子目录依赖
yarn bootstrap
```

## 项目文档

```bash
# 开发态
yarn dev

# 部署态
yarn build
```

## 组件开发

```bash
# 测试
yarn test

# 构建
yarn build:file

# 发布
# yarn publish
```
